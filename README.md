# 🏥 Медицинский статистический калькулятор для УЗД

## 📊 Описание
Веб-приложение для автоматического расчёта статистической значимости в медицинских исследованиях. Разработано специально для врачей ультразвуковой диагностики.

## ✨ Возможности
- **Точный критерий Фишера** — для качественных данных (таблицы 2×2)
- **U-критерий Манна-Уитни** — для количественных данных
- Автоматическая интерпретация результатов
- Сохранение истории расчётов
- Готовые форматы для диссертации

## 🚀 Демо
Приложение доступно онлайн:  
👉 [uzi-calculator.streamlit.app](https://uzi-calculator-akopyan.streamlit.app)

## 📋 Как использовать
1. Выберите тип анализа в боковой панели
2. Введите данные (качественные или количественные)
3. Нажмите кнопку расчёта
4. Получите p-value и интерпретацию

## 🛠 Технологии
- Python 3.10
- Streamlit
- Pandas, NumPy
- SciPy (статистика)
- Plotly (визуализация)
